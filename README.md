Landing Page Project (Level 1. Task 1)

📌 Overview

This Landing Page Project is a webpage that I made for a beginner-level web development task. The Landing Page Project is supposed to look nice and be easy to understand. I used HTML and CSS to make the Landing Page Project.

The main goal of the Landing Page Project is to learn about how to design the layout of a webpage and how to make things line up and look good. I also wanted to practice using the ideas of frontend development in the Landing Page Project.

🚀 Features

* The Landing Page Project has a layout that changes size depending on the screen

* The Landing Page Project has an modern look

* The Landing Page Project has the parts like a header and a footer

* I used the HTML elements to make the Landing Page Project

* I used CSS to make the Landing Page Project look nice with colors and spacing and everything

* The code, for the Landing Page Project is easy to understand

🛠️ Technologies Used

HTML5 is what I used to make the basic structure of the Landing Page Project

CSS3 is what I used to make the Landing Page Project look nice and to design the layout

📚 What I Learned

I learned how to make webpages that're easy to understand using HTML

I learned how to make things look nice with CSS

I learned how to make things line up properly and how to use space effectively

I learned how to pick colors that go well together

I learned how to keep things from overlapping in the Landing Page Project
